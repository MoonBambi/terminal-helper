🚀 核心技术栈方案
1. 容器层：Electron
理由： VS Code 的底层框架。它允许你使用网页技术构建桌面应用，并能直接调用底层操作系统的 API（如执行 shell 命令）。

优势： 社区生态最丰富，遇到任何终端执行的问题都能找到成熟的解决方案。

2. 构建工具：Electron-Vite
理由： 专门为 Electron 优化的构建工具，开箱即用，支持热更新（修改代码后界面立即刷新），开发体验极佳。

3. 前端框架：Vue 3 (或 React)
理由： 响应式系统非常适合处理“执行中”、“成功”、“失败”这类状态切换。

状态管理： 使用 Pinia 存储你的卡片数据和集合逻辑。

4. 视觉与交互 (IDE 风格的关键)：
样式控制：Tailwind CSS（必选）。它能通过简单的类名（如 border-[1px] border-white/10）实现 IDE 那种精细的边框和间距。

图标库：Lucide Vue。风格极简、线条纤细，非常符合 Antigravity 的审美。

组件库：Shadcn UI (Vue版)。这是目前最火的“高级感”组件库，它的暗色模式几乎是为 IDE 风格量身定制的。

5. 核心功能实现：
命令执行： Node.js 原生 child_process.spawn。支持流式输出，能让日志像真正的终端一样一行行滚出来。

数据存储： electron-store。简单可靠，将卡片和集合以 JSON 格式存在用户本地路径。

拖拽排序： vuedraggable (基于 Sortable.js)。实现卡片在集合中的自由排序。
