# Terminal Helper 项目笔记

## 项目结构
```
Terminal_helper/
├── data/
│   └── sample.json                 # 示例数据
├── src/
│   ├── main/                       # Electron 主进程
│   │   ├── index.js
│   │   ├── ipcHandlers.js           # IPC：执行命令 / 导入导出 / 运行日志等
│   │   └── store.js                 # JSON 数据持久化
│   ├── preload/                    # 桥接层
│   │   └── index.js                 # 暴露给前端的 API
│   ├── renderer/                   # 前端界面 (Vue 3)
│   │   ├── App.vue
│   │   ├── main.js
│   │   ├── style.css
│   │   ├── index.html
│   │   ├── assets/
│   │   ├── components/
│   │   └── modules/
│   │       ├── cards/
│   │       ├── collections/
│   │       └── terminal/terminalStore.js
│   └── stores/                     # 预留/历史目录
├── package.json
├── vite.config.js
├── tailwind.config.cjs
└── postcss.config.cjs
```

数据文件实际落地位置：
- `%APPDATA%/terminal-helper/data/terminal-helper.json`（Windows 对应 `C:\Users\<user>\AppData\Roaming\terminal-helper\data\terminal-helper.json`）
- 首次启动会从 `data/sample.json` 初始化

## 技术栈
- Electron 41（桌面端壳）
- Vite 8 + @vitejs/plugin-vue（构建）
- Vue 3（渲染层）
- Pinia（状态管理）
- Tailwind CSS（样式）
- lucide-vue-next（图标）
- vuedraggable（拖拽排序能力）
- Node.js child_process（命令执行）
- iconv-lite（Windows 输出编码解码）
- xterm + xterm-addon-fit（前端内嵌终端）
- node-pty（主进程 PTY 会话）

## 项目核心功能
- 可视化卡片式终端命令管理
- 卡片集合管理与串行执行
- 任务编排（串行运行多个集合）
- 终端页面多会话展示与交互
- 历史运行记录查看
- 本地 JSON 持久化与导入导出
- 可自定义 CMD/PowerShell/Bash 路径

## 目前已有功能
- 卡片管理
  - 新建 / 编辑 / 复制 / 删除
  - 搜索（命令 / 描述 / 关键词）
  - 运行终端选择（CMD / PS / Bash）
- 集合管理
  - 新建 / 编辑 / 复制 / 删除
  - 集合内支持重复卡片
  - 集合顺序可编辑（运行顺序数字气泡）
  - 集合列表支持搜索
  - 集合管理页支持导入/导出
  - 集合列表区域独立滚动
- 执行与日志
  - 单卡执行与集合串行执行（同一终端会话）
  - 集合日志面板改为“历史运行记录”（状态、开始/结束时间、耗时）
  - 任务日志面板改为“任务历史记录”（状态、开始/结束时间、耗时、进度）
  - 运行中可停止（按钮变为红色停止图标）
  - 集合详情固定“停止”卡片
  - 可配置停止命令（终端类型 / 是否新终端执行）
  - 停止支持进程树终止
  - 停止后保留已输出内容（减少尾部输出丢失）
  - 集合历史与任务历史均支持扫把按钮一键清理
- 终端页面
  - 左侧导航新增“终端”页面
  - 首次打开无会话时显示空状态
  - 运行集合后按集合 shell 自动创建终端 tab
  - 运行任务时按每个集合 shell 自动创建终端 tab 并串行运行
  - 运行结束后可在同一 tab 手动继续输入
  - 底部新增停止 / 清除连接圆形按钮（同集合页风格）
  - tab 支持 `X` 关闭（断开连接并移除标签）
  - tab 区改为单行横向滚动，滚轮可左右滚动，左右边缘渐变遮罩
  - 切换到其他页面后终端内容保持不变
- 导入导出
  - 全量导入/导出 JSON
  - 集合内导入/导出（仅集合所含卡片）
- 设置
  - CMD / PowerShell / Bash 路径设置
  - 路径检测与应用
- UI
  - 窗口操作行左侧加入品牌区（像素风 Logo + `DumbOx`）
  - Logo 支持点击旋转缩放动画
  - 3 列固定网格卡片
  - 底部浮动操作按钮
  - 抽屉式新建/编辑表单
  - 日志面板可折叠
  - 导航“全部卡片/集合管理/任务/终端/设置”统一图标风格
  - 任务卡片支持进度填充背景
  - 任务每完成一个集合，中心位置按顺序弹出圆形完成徽章（成功为勾、失败/停止为叉）
  - 任务历史面板样式对齐集合历史面板，固定在右下角，支持折叠
